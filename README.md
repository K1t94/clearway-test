# ClearwayTest

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.0.2.

## Development server

To start a local development server, run:

```bash
ng serve
```

Once the server is running, open your browser and navigate to `http://localhost:4200/`. The application will automatically reload whenever you modify any of the source files.

## Тест-кейсы

Чтобы открыть документ со списком, нужно ввести в адресной строке `http://localhost:4200/view/${documentId}`

documentId может быть любой строкой, это не имеет значения

Чтобы добавить аннотацию к изображению, нужно нажать на кнопку "Добавить аннотацию" и после 2 раза кликнуть в любой области любого изображения

Также реализовал немного "отсебятины", потому что были похожие задачи на текущем месте работы связанные с редактором текста, и захотелось продублировать сюда)
